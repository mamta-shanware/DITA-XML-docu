How to create and manage tasks

Here you will get instructions about how to create and manage tasks in TaskOrbit.

To successfully create and manage tasks in TaskOrbit:
  • You must first be logged into the application.
  • You must have the appropriate user permissions to create or edit tasks within the selected project.
  1. Open a project and click “+ New Task”.
  2. Enter a clear task title.
  3. Add a description with context or links.
  4. Assign a due date and responsible teammate.
  5. Add checklist items for sub-steps.
  6. Attach files or add tags.
  7. Click Create to save.
