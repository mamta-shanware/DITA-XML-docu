How to create custom roles

Here you will get instructions about how to create custom roles.

To successfully create custom roles:
  • You must be logged into the application.
  • Must have admin-level access to create and manage roles and permissions.
  1. Go to Roles > Create New Role.
  2. Name the role (e.g., Freelancer).
  3. Set permissions (e.g., view only, no delete access).
  4. Assign role to external users.

Freelancers see only their tasks, can't modify others, and can’t view sensitive documents—perfect for working with contractors securely.
