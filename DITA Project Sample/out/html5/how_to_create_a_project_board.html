How to create a project board

Here you will get instructions about how to create a project board.

To successfully create a project board:
  • You must be logged into the application.

  • You must have permission to create new projects (typically for team leads or project managers)

  1. Click Projects > New Project.
  2. Name your project and choose a view (Board, List, Calendar).
  3. Add workflow stages (columns like “To Do”, “In Progress").
  4. Invite team members and assign roles.
  5. Start adding tasks manually or from a template.

As a product manager, you create:

  • Project: “Q3 Product Launch"
  • Columns: Planning → Design → Development → QA → Launch
  • Team Members: Dev, Design, QA, Marketing
